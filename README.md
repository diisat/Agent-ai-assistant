#Agent AI Assistant

**Agent-ai-assistant** is a dual-agent system designed to assist with **marketing** and **sales** tasks.  
It integrates **Groq-powered AI agents** with **FastAPI** for backend communication and **Streamlit** for an interactive frontend interface.  
The project also includes **Jupyter notebooks** for testing and analysis.

---

## Features

- **Two specialized AI agents**
  - **Marketing Agent** — focuses on content strategy, audience targeting, and campaign optimization.
  - **Sales Agent** — assists with lead management, customer insights, and sales strategies.

- **FastAPI Backend**
  - Provides RESTful endpoints for communication between the UI and agents.
  - Handles agent response generation using Groq inference.

- **Streamlit Frontend**
  - Simple, interactive chat interface for real-time communication with the agents.

- **Jupyter Testing**
  - Contains notebooks for experimenting and validating agent behavior.

- **Data Management**
  - Configuration and experimental data stored in structured files (`.xlsx`, `.json`).
